# Todo Application

할 일 관리를 위한 React 애플리케이션입니다.

## 🚀 주요 기능

- [ ] 할 일 추가/삭제
- [ ] 할 일 완료 체크
- [ ] 카테고리 분류
- [ ] 로컬 스토리지 저장

## 💻 기술 스택

- React
- Vite
- Tailwind CSS
- shadcn/ui

## 🛠️ 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
